Quem sou eu?

Nos últimos meses, mergulhei no fascinante mundo do desenvolvimento web, focando em três tecnologias essenciais: HTML, CSS e JavaScript. Minha jornada começou com o HTML, que é a espinha dorsal de qualquer página da web. Aprender a estruturar informações com tags e elementos foi como aprender a construir uma casa; cada parágrafo, imagem e link é uma parte fundamental que traz vida ao projeto. Com o HTML dominado, o próximo passo foi o CSS. Aqui, pude dar asas à minha criatividade! Com o CSS, aprendi a estilizar minhas páginas, escolhendo cores, fontes e layouts. A primeira vez que vi minha página ganhar um visual atraente foi uma sensação incrível. O CSS me ensinou que a apresentação é tão importante quanto o conteúdo, e a combinação de ambos pode transformar uma ideia simples em algo impactante.
